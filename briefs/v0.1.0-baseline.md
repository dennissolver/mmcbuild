## What this tag represents

This is the baseline snapshot of the MMC Build MVP at the conclusion of
Stages 0–5. All six modules were built near-simultaneously ahead of the
original 14-week schedule. This tag is applied retroactively to mark the
state of the platform at the first live client review meeting (1 April 2026).

---

## What is live at this tag

### Infrastructure
- Stack: Next.js (App Router) + Vercel + Supabase (Sydney region) + Inngest
- Auth: Supabase auth with row-level security, passing checks
- RBAC: org owner / admin / member roles enforced
- Audit logging: active across all user actions
- LLM routing: Open Router (multi-model, cost-balanced) with token burn monitoring
- Secrets: all in Vercel environment variables, nothing client-exposed
- AU data residency: confirmed (Supabase Sydney)
- AI evals: running for drift and hallucination detection
- AWS migration path: encoded in DB migrations, not yet executed

### Module: MMC Comply
- Plan upload (PDF, DWG, DXF, PNG, JPG)
- NCC 2022 compliance analysis engine (20+ requirements, Parts B/C/F/J)
- Compliance report with status indicators (Compliant / Conditional / Non-Compliant / N/A)
- Criticality levels (Critical / Major / Minor)
- Evidence with Australian Standards references (30+ standards)
- PDF export of compliance report

### Module: MMC Build
- Plan upload with shared plan storage (cross-module)
- Material analysis engine (12 MMC materials)
- 3D visualisation viewer
- Design optimisation suggestions
- PDF export of build report

### Module: MMC Quote
- Cost estimation from seeded industry-average benchmark data
- Manufacturer pricing database (10+ AU manufacturers)
- Itemised quote with GST, bulk discounts, labour
- PDF and text export

### Module: MMC Direct
- Verified directory of 25+ AU professionals and suppliers
- 9 categories, state and keyword filtering
- ABN lookup built; QLD licence verification API in progress
- Profile cards with MMC experience, certifications, contact details

### Module: MMC Train
- 6 self-paced training modules (Prefab Assembly, 3D Printing, CLT, LSF, Modular PM, Safety)
- Slide-by-slide viewer (video + text)
- Quiz system (5 questions, 80% pass threshold, unlimited retakes)
- Certificate generation on pass
- Progress persistence via browser LocalStorage

### Module: Billing
- Stripe integration (test mode)
- Subscription placeholder (2-month free trial → paid)
- Tier logic: pending admin platform stability

### Dashboard
- Admin/User view toggle (UAT-ready for Karen and Karthik)
- Admin view: platform config, cost rates, knowledge bases, operations
- User view: module dashboard with subscription status
- Project creation dialog with property intelligence (Mapbox geocoding, lot size, zoning, wind/climate/BAL, use-case assessment)
- Shared plan storage across Comply, Build, Quote modules

### Known issues at this tag
- Users can access Quote without first creating a project (dashboard onboarding flow not yet guided)
- Material/system selection in Build module is automated without user toggle control
- Directory uses open listing model; admin approval workflow not yet built
- Report versioning (tracking iterations) not yet implemented
